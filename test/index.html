Scroll within this box, or click/type anywhere on the page. The extension's content script listens for these events and will skip the next scheduled reload if you interacted recently enough.

Serve this page over HTTP (e.g. python3 -m http.server from the test/ folder) and add a rule in the extension for the resulting host (e.g. localhost). The reload count and log persist across reloads via localStorage, scoped per origin/port. Use "Reset counter" to start a clean test run.
